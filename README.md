# 🤖 Bot de Soporte Técnico SUVE (Variante KS2)

Esta es una variante del bot de Telegram para la gestión automatizada de tickets de soporte técnico.

## 🛠️ Tecnologías
- Python 3.10+
- python-telegram-bot v20+
- SQLAlchemy
- Integración GLPI REST API

## 🚀 Instalación y Configuración
Por favor refiérase al repositorio principal [bot_soporte](https://github.com/Gabodat/bot_soporte) para instrucciones detalladas de despliegue, arquitectura y uso, ya que este repositorio comparte la misma base de código.
